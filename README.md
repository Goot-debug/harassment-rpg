# harassment-rpg
# ハラスメント回避RPG 〜新米職員の10日間〜

地方自治体職員向けのハラスメント防止教育コンテンツです。  
新米行政職員として10の場面に遭遇し、会話の選択肢を選びながらハラスメントについて学べます。

---

## 📖 概要

- **形式**：スタンドアロンHTML（サーバー・インターネット接続不要）
- **対象**：地方自治体職員（主に新規採用・若手職員向け）
- **プレイ時間**：約10〜15分

### 登場するハラスメント類型

| シーン | テーマ |
|--------|--------|
| 1日目 | コミュニケーション全般 |
| 2日目 | セクハラ・プライバシー配慮 |
| 3日目 | エイジハラスメント |
| 4日目 | カスハラ対応 |
| 5日目 | マタハラ・育児配慮 |
| 6日目 | パワハラ・指導 |
| 7日目 | アルハラ |
| 8日目 | プライバシー・SNS |
| 9日目 | パワハラ・時間外 |
| 10日目 | メンタルヘルス・心理的安全性 |

---

## 🚀 使い方

1. `https://raw.githack.com/Goot-debug/harassment-rpg/refs/heads/main/harassment-rpg-html.html` をブラウザで開く
2. 「出勤する」ボタンでスタート
3. 各シーンで A・B いずれかの選択肢を選ぶ
4. 全10シーンをクリアしてエンディングを確認する

> ⚠️ リスクのある選択を **3回以上** 選ぶとバッドエンドになります。

## 🖼️ シーン画像の設定方法

`index.html` 内の各シーンデータにある `imageUrl` に画像のパスを指定してください。

```js
// 例：imagesフォルダ内の画像を使う場合
imageUrl: "images/scene01.png"
```

---

## 🛠️ カスタマイズ

`index.html` はすべて1ファイルで完結しています。  
シーンの内容（状況・セリフ・選択肢・解説）は `SCENES` 配列を直接編集することで変更できます。

```js
const SCENES = [
  {
    id: 1,
    title: "シーンタイトル",
    situation: "状況説明",
    npc_dialogue: "登場人物のセリフ",
    choice_risky: "リスクのある選択肢",
    choice_safe: "配慮ある選択肢",
    result_risky: "リスク選択の解説",
    result_safe: "安全選択の解説",
    imageUrl: "images/scene01.png"
  },
  // ...
]
```

バッドエンドの閾値（デフォルト：3回）も変更可能です。

```js
const BAD_THRESHOLD = 3; // この数値を変更
```
